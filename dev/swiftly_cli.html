class swiftly.cli.CLI(args=None, stdin=None, stdout=None, stderr=None)[source]¶

Bases: object

An instance of a command line interface client to Swift. After construction, a call to main() will execute the command line given. No args will output help information.

Parameters:
  • args – The command line arguments to process.
  • stdin – The file-like-object to read input from.
  • stdout – The file-like-object to send output to.
  • stderr – The file-like-object to send error output to.
main()[source]¶

Process the command line given in the constructor.
